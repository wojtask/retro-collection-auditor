RetroCollection Auditor

Command Line Interface Tool

This application is a local CLI script. The web interface is deprecated.

Usage Instructions

# 1. Standard (Gamelists inside ROMs folder)

node scan.js "D:/Path/To/Roms" > report.csv

# 2. Split (Gamelists separate from ROMs)

// Use this if you have a separate .emulationstation/gamelists folder

node scan.js "D:/Roms" "C:/Users/User/.emulationstation/gamelists" > report.csv

Ensure Node.js is installed. Run npm install before use.
